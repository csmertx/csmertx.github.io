## Firefox

- ```CTRL``` + ```P```

    - Save to PDF

## Chome

- ```CTRL``` + ```P```

    - Save to PDF

## wkhtmltopdf

- ```wkhtmltopdf input.html output.pdf```

    > Additional options via CLI not available through browsers
