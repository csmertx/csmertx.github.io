# Tar: File storage compression for GNU/Linux

## Decompress

- ```.tar.gz```

    - ```tar -xvzf file.tar.gz```

- ```.tar.xz```

    - ```tar -xvf file.tar.xz```

- ```.tar.bz2```

    - ```tar -xjvf file.tar.bz2```

## Compress

- ```.tar.gz```

    - ```tar -czvf file.tar.gz directory_or_file```

- ```.tar.xz```

    - ```tar -cvf file.tar.xz directory_or_file```

- ```.tar.bz2```

    - ```tar -cjvf file.tar.gz directory_or_file```
