## Recreate session layouts

- Build a single window layout and run: ```tmux list-windows```

- Copy it and use it with ```tmuxinator```, etc.

## Install Tmuxinator

- ```apti ruby-full```

    > Or see links [below](#resources) for a comprehensive guide

- ```gem install tmuxinator```
