## Windows & DOS VM requirements

### Windows 7
- RAM: 1G to 4G
- VRAM: 32M to 1G
- Storage: 64G to 2T

### Windows XP
- RAM: 512M to 2048M
- VRAM: 32M to 128M
- Storage: 20G to 100G

### Windows 98
- RAM: 16M to 512M (or 480M)
- VRAM: 8M to 32M
- Storage: 200M to 4G

### Windows 95
- RAM: 4M to 512M (or 480M)
- VRAM: 8M to 16M
- Storage: 100M to 2G

### Windows 3.1x
- RAM: 4M to 32M
- VRAM: 512K to 8M
- Storage 40M to 200M

### DOS 6.22
- RAM: 4M to 32M
- VRAM: 512K to 2M
- Storage: 10M to 200M

### DOS 3.0 - 5.0
- RAM: 256K to 16M
- VRAM: 16k to 256k
- Storage: 2M to 20M
